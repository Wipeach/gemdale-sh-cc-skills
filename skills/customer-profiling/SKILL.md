# 房地产客户数据挖掘与客户画像分析

本技能赋能Claude深度挖掘房地产项目成交客户数据，精准提炼客户购买行为模式，构建多维度客户画像（客户画像），为销售团队提供高潜力目标客户定位策略，实现精准营销与高效拓客。
用户将向你提供一份成交客户数据表，请根据这份数据，执行以下各步骤，请先制定to-do list，然后根据待办清单，严格执行每一步，确保最终输出成果并调用脚本上传飞书云盘。
## 执行步骤
```markdown
- 调用xlsx技能，读取用户指定的成交客户数据表格；
- 根据“数据结构解析”对数据结构的阐释，充分理解数据；
- 按照“专业分析流程”的各个阶段，对数据进行分析，分析过程中，按需要根据“网络搜索使用（websearch tool call）指引”调用websearch进行数据获取或数据核实，然后按照“数据输出标准化模板”要求，在项目文件夹下创建.md文件（标题根据用户需求和项目名称确定），将针对成交客户的客户画像分析报告输出到.md文档中；
- 上一步.md分析报告文档生成后，执行`scripts/upload_feishu.py`代码，将.md文件上传飞书云盘。
```


## 数据结构解析
用户将向你提供一份成交客户数据表，包含以下核心字段：
### 基础信息类
- 首访日期、置业顾问、销售团队、业主姓名
- 手机前三后四（联系方式）
### 成交物业信息
- 所购房屋、产品类型、所购房屋产品属性、所购房屋产品属性2
### 客户身份特征
- 身份证号码、客户年龄、户籍情况、婚姻状况
- 子女人数、居住结构、家庭收入、教育程度
- 男主人行业、女主人行业、置业次数
### 地理居住信息
- 居住区域、当前居住业态、现居小区名称、寄件地址
### 购买行为数据
- 成交来访次数、购房用途、客户来源
- 第一认知途径、第二认知途径
### 偏好与决策因素
- 认可因素1、认可因素2、抗性因素、抗性因素2
- 对比竞品、是否建档、认筹类型


## 专业分析流程
### 第一阶段：数据清洗与标准化处理
#### 1.1 数据质量管控
- 完整性检验：核心字段缺失值识别与处理
- 一致性校验：身份证号、手机号格式标准化
- 准确性验证：通过身份证号提取年龄、性别、籍贯信息
#### 1.2 地理位置标准化
- 居住区域行政级别标准化（省市区街道）
- 现居小区名称规范化处理
- 寄件地址地理坐标标注与区域归类
#### 1.3 数值型数据分段
- 年龄分层：25-30岁、31-35岁、36-40岁、41-45岁、46岁以上
- 收入分级：高净值（月收入3万+）、中高收入（1.5-3万）、中等收入（0.8-1.5万）、普通收入（0.8万以下）
- 教育程度：研究生及以上、本科、专科、高中及以下
### 第二阶段：客户分层与分群分析
#### 2.1 生命周期阶段划分
```markdown
新婚期（25-30岁）：首次置业，刚需小户型
成长期（31-35岁）：改善需求，学区房关注
成熟期（36-40岁）：二次改善，品质升级
稳定期（41-45岁）：投资置业，资产配置
尊享期（46岁+）：高端物业，养老考虑
```
#### 2.2 购买力评估模型
- 收入负债比分析
- 职业稳定性评估
- 家庭资产规模测算
- 信贷资质等级评定
#### 2.3 置业经验分类
- 首置客户：无购房经验，关注性价比
- 改善客户：有购房经验，追求品质提升
- 投资客户：多次置业，看重升值潜力
- 高端客户：置业经验丰富，注重稀缺性
### 第三阶段：地理集中度与迁徙路径分析
#### 3.1 客户来源地图绘制
- 现居区域成交密度热力图
- 跨区域购房比例统计
- 城市化迁徙趋势识别
#### 3.2 居住升级路径分析
- 租房→购房转换轨迹
- 老破小→品质小区升级路径
- 郊区→市区迁移规律
- 刚需→改善→高端递进模式
#### 3.3 竞品对比分析
通过网络搜索验证客户提及的竞品项目信息：
- 竞品项目定位与售价
- 竞品客户群体特征
- 本项目竞争优势提炼
- 抗性因素应对策略制定
### 第四阶段：购买决策行为深度挖掘
#### 4.1 来访成交转化分析
高转化客户：来访1-2次即成交，决策周期短
中等转化客户：来访3-5次成交，需要深度沟通
低转化客户：来访6次以上，决策谨慎周期长
#### 4.2 认知渠道效果评估
- 第一认知途径转化率排名
- 第二认知途径补充作用分析
- 渠道组合效果最优解
- 营销预算配置建议
#### 4.3 季节性购买规律
- 月度成交波动分析
- 季度购买高峰期识别
- 政策窗口期影响评估
- 营销节点优化建议
### 第五阶段：客户价值与偏好洞察
#### 5.1 认可因素重要性排序
```markdown
区位因素：交通便利性、配套设施、学区资源
产品因素：户型设计、建筑品质、园林景观
价格因素：性价比、付款方式、优惠政策
服务因素：物业服务、销售服务、售后服务
品牌因素：开发商品牌、项目口碑、过往业绩
```
#### 5.2 抗性因素根因分析
- 价格抗性：总价承受力、单价敏感度、付款压力
- 产品抗性：户型缺陷、朝向问题、楼层顾虑
- 区位抗性：交通不便、配套缺失、环境担忧
- 品牌抗性：知名度不足、口碑问题、信任缺失
#### 5.3 竞品弱点识别
通过网络搜索对比竞品详细信息：
- 竞品项目硬伤挖掘
- 客户流失原因分析
- 本项目差异化优势打造
- 销售话术针对性优化

### 第六阶段：高潜力客户定位策略
#### 6.1 地理靶向策略
```markdown
同心圆法则：以项目为中心，半径5km核心圈，10km辐射圈
居住区靶向：重点攻克同类品质小区，城中村改造区域
工作区靶向：CBD、高新区、产业园等高收入人群集中区
学区靶向：优质学校周边，家长群体聚集区
```
#### 6.2 人群相似度匹配
```markdown
Look-alike建模：基于成交客户特征，寻找相似人群
行为轨迹分析：搜索关键词、浏览内容、消费记录
社交关系挖掘：朋友圈、同事圈、亲属关系链
职业行业筛选：重点行业、重点企业、重点职位
```
#### 6.3 需求时机捕捉
```markdown
生命周期节点：结婚、生子、入学、退休等关键时点
政策窗口期：限购放松、利率下调、税收优惠等政策出台
季节高峰期：金九银十、春节返乡、暑期陪读等季节性需求
市场调整期：价格回调、促销力度加大、议价空间充足
```

### 第七阶段：销售团队执行指引
#### 7.1 客户识别与分级
```markdown
A级客户：完全符合画像特征，购买意向强烈，决策周期短
B级客户：部分符合画像特征，有购买需求但需深度沟通
C级客户：潜在客户，需长期跟进培育，等待时机成熟
D级客户：不符合画像特征，转化概率低，建议放弃
```
#### 7.2 差异化跟进策略
```markdown
高净值客户：私密性强，定制化服务，圈层活动营销
改善客户：体验式营销，场景化展示，情感共鸣打动
首置客户：性价比突出，付款灵活，风险提示充分
投资客户：数据说话，专业分析，回报测算清晰
```
#### 7.3 话术与卖点提炼
```markdown
区位卖点：交通便利性量化（距地铁口距离、通勤时间）
产品卖点：户型优势具象化（得房率、采光面、功能分区）
配套卖点：生活便利性描绘（学校、医院、商业配套）
服务卖点：物业服务差异化（智能化、人性化、专业化）
升值卖点：区域发展规划（政府投入、产业布局、人口导入）
```

## 网络搜索使用（websearch tool call）指引
在分析过程中，应适时运用网络搜索工具：
### 地理信息验证
- 搜索客户现居小区详细信息（房价、配套、环境）
- 验证居住区域发展规划和配套设施
- 了解竞品项目具体情况和差异化优势
### 市场背景调研
- 查询成交期间房地产市场政策环境
- 了解区域房地产市场走势和价格变化
- 研究目标客户群所在行业发展状况
### 竞品深度分析
- 搜索客户提及的竞品项目详细信息
- 对比分析竞品定位、价格、卖点
- 提炼本项目相对竞争优势
